# Install Instinct8 Agent

## Quick Install

```bash
pip install instinct8-agent
export OPENAI_API_KEY="your-key"
instinct8 "Hello!"
```

## That's It!

Users can now install Instinct8 Agent directly from PyPI with a single command.
