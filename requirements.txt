# Instinct8 Agent - Dependency Management
#
# This project uses pyproject.toml as the source of truth for dependencies.
# For different installation scenarios, use:
#
# MINIMAL INSTALL (core CLI functionality only):
#   pip install -e .
#
# DEVELOPMENT (includes testing tools):
#   pip install -e ".[dev]"
#
# EVALUATION (includes LLM providers and metrics):
#   pip install -e ".[eval,metrics,data]"
#
# FULL INSTALL (all dependencies including heavy ML frameworks):
#   pip install -e ".[all]"
#
# For CI/CD or automated environments, you can install with all dependencies:

-e .[all]
